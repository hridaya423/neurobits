# Neurobits Website

This directory contains the Neurobits marketing website built with Next.js.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Key Files

- `app/page.tsx`: homepage and scroll sections
- `app/layout.tsx`: metadata, title, social tags, icons
- `public/icon.png`: site icon and social preview image

## Quality Checks

```bash
npm run lint
```
